## 🚀 快速开始

### 1. 克隆仓库

```bash
打开本地 Git Bash 或终端，定位到您期望存放项目的本地工作目录（例如 /d/workspace），执行克隆命令将您 Fork 后的个人仓库下载至本地：
git clone https://github.com/BISTU-OSSD/practice-group-9.git
/git clone https://github.com/您的GitHub用户名/practice-group-9.git
cd practice-group-9
2.本地编辑
打开 src/index.html
3.推送代码与一键 Pages 发布
提交修改：在本地终端依次运行以下命令，将定制后的简历内容提交并推送到您个人的 GitHub 仓库：
git add .
git commit -m "docs: customize personal resume content and profile photo"
git push origin main

配置 GitHub Pages 服务：
登录您的个人 GitHub 网页端，进入您刚才 Fork 的 practice-group-9 仓库。
点击上方导航栏的 Settings，在左侧菜单栏中点击 Pages。
在 Build and deployment 区域，将 Branch 设置为 main，目录保持选择默认的 / (root)，然后点击 Save。

获取在线链接：等待大约 1 分钟，刷新该页面，顶部会显式生成您的专属简历链接。直接点击该链接，并在网址末尾追加 /src/index.html 即可进行线上浏览。
您的专属在线地址格式为：https://您的GitHub用户名.github.io/practice-group-9/src/index.html
